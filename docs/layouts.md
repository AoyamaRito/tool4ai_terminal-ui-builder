## Best Practices

1. **Height Guidelines**:
   - Minimum: 15 rows
   - Optimal: 20-30 rows
   - Maximum: 50 rows

2. **Width**: Always 80 characters (fixed)

3. **Content**:
   - Use abbreviations for long text
   - Keep labels concise
   - Prioritize important information

4. **ASCII Safety**:
   - Stick to basic ASCII characters
   - Test in different terminals
   - Avoid Unicode symbols
